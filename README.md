# Ayesha Aziz Muhammad - Virtual CV & Portfolio

Welcome to the codebase for my **Virtual CV**! This dynamic portfolio website serves as an interactive representation of my professional resume, built to showcase my technical skills, creativity, and aesthetic sense.

## Project Functionalities

- **Tech Girly / Y2K Aesthetic**: A unique, bold design featuring neon colors, brutalist shadows, and custom fonts to stand out from typical generic templates.
- **Responsive Layout**: Designed to look great on desktop, tablet, and mobile browsers.
- **Interactive Elements**:
  - Smooth scrolling navigation.
  - Intersection Observer-based fade-in animations as you scroll down the page.
  - A dynamic, infinitely scrolling marquee to highlight my technical skills.
  - Hover effects on cards and buttons.
- **Comprehensive CV Sections**: Includes an About Me section, Education history, Certifications, a Projects showcase, Skills, and Contact Information.
- **Downloadable CV**: A button that allows recruiters or visitors to directly download my professional CV in PDF format.

## Setup Instructions

Since this is a static website built with vanilla HTML, CSS, and JavaScript, no complex build process is required to view or run it.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ayeshaaziz04/CMPG323-CV.git
   ```

2. **Run the project:**
   Simply open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Edge, Safari).
   - Alternatively, if you have a local development server installed (like VS Code's Live Server or Python's HTTP server), you can serve the directory to view it on `localhost`.

3. **Add the CV PDF:**
   For the "Download CV" button to work correctly, ensure you have a copy of the actual PDF named `Ayesha_CV.pdf` inside the `assets/` directory. (A placeholder file is included by default).
