# My Virtual CV

Hey there! 👋 This is the source code for my interactive virtual CV and portfolio website. I built this for my CMPG 323 module to showcase my technical skills, education, and some of the projects I've been working on.

I decided to go with a bright, Y2K "tech girly" aesthetic because I wanted something a little more fun and creative than a standard corporate resume template.

### What's inside:
- **HTML, CSS, and Vanilla JS:** Kept it simple with no heavy frameworks.
- **Responsive design:** Works nicely on both mobile and desktop.
- **Custom animations:** Added some smooth scrolling, a scrolling skills band, and fade-in effects as you navigate the page.

### Want to check it out locally?
Just clone the repo and open `index.html` in your browser. No complicated setup required!

```bash
git clone https://github.com/ayeshaaziz04/CMPG323-CV.git
```

*Note: The actual PDF download button points to `assets/Ayesha_CV.pdf`.*
